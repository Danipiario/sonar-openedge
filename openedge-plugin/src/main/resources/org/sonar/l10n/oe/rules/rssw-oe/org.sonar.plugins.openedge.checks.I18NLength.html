This rule raises an issue whenever if finds a LENGTH / OVERLAY /SUBSTRING function call which doesn't provide the 'type' parameter.

Noncompliant Code Example

  /* Will display 1 */
  MESSAGE LENGTH('乗').

Compliant Solution

  MESSAGE LENGTH('乗', 'CHARACTER'). /* Only one character */
  MESSAGE LENGTH('乗', 'COLUMN').    /* Which takes 2 columns to display */
  MESSAGE LENGTH('乗', 'RAW').       /* And 3 bytes in UTF-8 */
